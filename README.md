# Project Features:

1. get -> api/todo/:email -> Get all todo articles
2. post -> api/todo -> Add todo articles in database
3. get -> api/todo/:id -> get todo articles in database with sepecific by id
4. delete -> api/todo/:id -> Delete todo by id

- Technologies use
  1.  cors
  2.  express js
  3.  mongoose
  4.  dotenv
